【「Local Media Viewer」の特徴】  
・ウェブブラウザのタブでローカルのフォルダツリーを表示し、その中から選択されたローカルメディアを表示します。  
・「Local Media Viewer」は、例えば、静止画を１枚ずつ順次表示したり、フォルダ内の複数の静止画を使ってスライドショーを開始したり、動画を字幕付きで再生したりできます。  
  
【「Local Media Viewer」のインストール】  
(1) Chromeの拡張機能の「デベロッパーモード」をONに設定する。  
(2) 「パッケージ化されていない拡張機能を読み込む」をクリックする。  
(3) この拡張機能が保存されているフォルダを選択する。  
  
【「Local Media Viewer」のオプションのセットアップ】  
(1) フォルダツリーを表示するデフォルトフォルダを設定する。  
(2) スライドショーの表示間隔（秒）と表示順序（昇順、降順、ランダム）を設定する。  
(3) 画像表示時にファイル名を表示するか、表示しないかを設定する。  
(4) 保存ボタンをクリックして設定を保存する。  
(5) 拡張機能のキーボードショートカットで、この拡張機能を使うためのショートカットキーを、例えば、control+L (Mac)/Alt+L (Win)に設定する。  
  
【「Local Media Viewer」を使う前のセットアップ】  
(1) この拡張機能のアイコンをクリックするか、もしくは、ショートカットキーを押すと、「Local Media Viewer」「Local Media Viewer」が新規タブに表示される。
(2) 初めて「Local Media Viewer」を使うときは、ローカルメディアにアクセスするための権限を与える。  
(3)「フォルダを選択」ボタンをクリックすると、フォルダツリーを表示するフォルダを切り替えることができる。  
  
【「Local Media Viewer」の使い方１】  
・静止画を閲覧する。  
(1) フォルダツリーの中から、目的の静止画のファイルを選択する。  
(2) 「◁／▷」ボタンをクリックすると、１つ前／１つ後の静止画を順次閲覧できる。  
  
【「Local Media Viewer」の使い方２】  
・スライドショーを開始する。  
(1) フォルダツリーの中から、目的の静止画のファイルを選択する。  
(2) 「スライドショー」ボタンをクリックすると、スライドショーが開始する。  
(3) 「スライドショー停止」ボタンをクリックすると、スライドショーが停止する。  
  
【「Local Media Viewer」の使い方３】  
・動画を再生する。  
(1) フォルダツリーの中から、目的の動画のファイルを選択すると、選択した動画が再生される。  
(2) 動画ファイルと同じ名前のSRT形式の字幕ファイルがあれば、字幕が表示される。  
(3) 字幕はプレーヤでオン／オフを切り替えることができる。  
  
【まとめ】  
・「Local Media Viewer」を使えば、ウェブブラウザからメディアブラウザに切り替えることなく、ウェブブラウザでローカルメディアを閲覧することができます。
